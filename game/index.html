DIAMONDS!

a match 3 or more HTML5 game prototype written in javascript & and pixie.js
select diamonds by clicking on them, reload the page to restart the game

Introduction

I coded this small game to learn some HTML5 game development with pixie.js. The code can be found on github.

26. feburary 2014 - first public release

The first public version of my DIAMONDS! HTML5 game prototype got released today. In the last two days, I nearly spent 10 hours to find one stupid bug which I erased yesterday finally. The most simple bugs are often the worst bugs to find. There might be some glitches left but overall the prototype just runs fine as far as I tested the code.
